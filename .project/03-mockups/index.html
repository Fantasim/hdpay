HDPay — Design Mockups

Self-hosted cryptocurrency payment tool. These mockups define the visual design system and screen layouts for the Svelte dashboard. Component styles (tokens.css, components.css) are designed for reuse in the actual build.

Component Gallery

  • All Components Colors, typography, buttons, forms, tables, badges, progress bars, alerts, and more

Screens

  • Dashboard Portfolio summary with total USD value, quick stats, balance breakdown table, and chart placeholder
  • Address Explorer Chain tabs, balance filters, paginated address table with copy buttons and export
  • Scan Control Unified scan panel with chain dropdown, active scan progress, and provider health status
  • Send / Sweep Wizard-style flow: select chain/token, preview fees, gas pre-seed, execute consolidation
  • Transaction History Filterable table with chain, direction, and token filters, explorer links, and status badges
  • Settings Network mode, scan defaults, transaction fees, display preferences, and danger zone actions
